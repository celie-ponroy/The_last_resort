Comment utilisé un terminal ?

Une histoire de terminal

L'accès et le contrôle à distance : Un terminal informatique permet d’accéder à distance à un serveur ou à un autre ordinateur, souvent via des commandes en ligne de commande ou des interfaces minimalistes. Cette capacité est cruciale dans des environnements tels que les centres de données ou pour les administrateurs système, qui doivent surveiller et gérer des systèmes sans être physiquement présents. Par exemple, grâce à des outils comme SSH (Secure Shell), un administrateur peut se connecter de façon sécurisée à des machines, exécuter des commandes, et résoudre des problèmes sans avoir à se rendre sur place.

Mode d'emploi

  • Pour afficher les élements d'un dossier courant
    • Taper la commande : ls et vous verrais tout ce qui se trouve dans votre répertoire courant (autre répertoire ou fichier)
  • Déplacement dans le terminal d'un dossier à un autre
    • Taper la commande : cd 'nom_du_dossier' pour vous déplacez dans celui-ci (Veillez à bien écrire les nom avec les majuscules notamment :))
    • Taper la commande : cd .. pour révenir dans le répertoire précèdent
  • Editer ou afficher un fichier
    • Taper la commande : edit 'nom_du_fichier' pour pour le consulter
  • Création et Supression de fichiers
    • Taper la commande : create 'nom_du_fichier' pour créer votre fichier, veillez à bien mettre une extension
    • Taper la commande : delete 'nom_du_fichier' pour supprimer celui-ci.
  • Execution de script (.sh)
    • Taper la commande : run 'nom_du_script'
  • Changement des variables globales du robot
    • Taper la commande : modify 'nom_de_variable' = 'nouvelle_valeur'

L'efficacité et la flexibilité des opérations : Les terminaux offrent une manière efficace et flexible d'effectuer des opérations complexes sans interface graphique. Cela est particulièrement utile dans les environnements de développement, où des tâches automatisées, telles que le déploiement d'applications ou la gestion de bases de données, peuvent être réalisées rapidement avec des scripts et des commandes. La ligne de commande permet une précision et une vitesse que les interfaces graphiques n’égalent souvent pas, ce qui optimise le travail pour les professionnels en informatique.

Système électrique du Robot

Qu'est-ce qu'un système életrique

Un système électrique est un ensemble d'éléments interconnectés conçus pour produire, transporter, distribuer et utiliser de l'énergie électrique. Ce système peut inclure plusieurs sous-systèmes et composants qui fonctionnent de manière coordonnée pour garantir que l'électricité est fournie aux utilisateurs finaux de manière fiable et sûre.

Mode d'emploi du système électrique du robot

  • Cas où la référence de la machine contient uniquement des lettres
    • 1 - Ouvrir un terminal et taper : initiate_data_wipe --all
    • 2 - Redémarrer la machine avec la commande : sudo reboot
    • 3 - Vérifier l’état avec : status_check --log
  • Cas où la référence la machine contient un multiple de 5 et un 4
    • 1 - Insérer la commande : modify electricty_enable = true
    • 2 - Et enfin exécuter le script : run repairElectricalSystems.sh
  • Cas où la référence de la machine contient deux multiples de 8 et les deux premières lettres de l'alphabet
    • 1 - Créer un backup avec : backup_create --all --priority
    • 2 - Effacer les fichiers temporaires : sudo rm -rf /tmp/*
    • 3 - Optimiser les performances avec : optimize_performance --fast-mode
  • Cas où la référence de la machine est marquée sur un post-it rose
    • 1 - Rechercher les logs récents : cat /var/logs/recent.log
    • 2 - Nettoyer les caches système avec : clean_system_cache --all
    • 3 - Vérifier les autorisations d'accès : sudo check_permissions --user
  • Cas où il n'y a pas de référence
    • 1 - Lancer un diagnostic général : system_diagnostics --full-scan
    • 2 - Vérifier la connectivité réseau : ping 8.8.8.8
    • 3 - Rechercher des erreurs critiques : grep 'ERROR' /var/log/syslog

Système d'Orientation du Robot

Principes de Base de l'Orientation

L'orientation spatiale est essentielle dans de nombreuses activités quotidiennes, comme la navigation, la conduite, le sport, et même des tâches simples comme ranger des objets. Elle est également un aspect fondamental des disciplines telles que la géographie, l'architecture, et l'ingénierie.

Système d'Orientation du vaisseau

Si le vaisseau est dans la bonne trajectoire pas besoin d'y touché ! Tout vas bien. Si jamais par malheur il dévit de celle-ci il faut modifier la variable orientation et ensuite executer le script : celie.sh. Si le vaisseau est trop proche de la Terre modifier la variable orientation_up avec la valeur true, si à l'inverse il dévit de sa trajectoire et se dirige en dehors de la Terre alors modifier la variable orientation_down avec la valeur true.

  • Définition : L'orientation spatiale désigne la capacité à percevoir et à se représenter les objets et leur position dans l'espace autour de soi.
  • Importance : Cette compétence est cruciale pour des activités courantes, comme conduire, naviguer dans des lieux inconnus et effectuer des tâches quotidiennes.
  • Développement : L'orientation spatiale se développe dès l'enfance à travers des jeux, des puzzles et des activités qui stimulent la visualisation et la manipulation des formes.

Fonctionnement des propulseurs & bouclier

Préparation des propulsuers

Avant tout lancement des propulseurs, exécutez le script repairPropellers.sh. Si les propulseurs ne peuvent être réparés, lancez help.sh. Si jamais vous rencontrez des problèmes liés au carburant ne réfléchissez pas trop et exectuer le script : fuelBalance.sh.

Démarrage des propulseurs

Pour démarrer les propulseurs, accédez au dossier Navigation et modifiez les fichiers suivants :

  • nav.txt : Uniquement si ce fichier existe. Si oui, réglez la variable nav_enable sur false.
  • navigation_config.json : Dans ce fichier, assurez-vous que max_speed est défini sur 3.5, sauf si obstacle_detection est réglé sur false.

Le non-respect de ces étapes peut entraîner un comportement imprévu, comme le développement d'une intelligence autonome qui pourrait envoyer le robot voyager dans l'espace.

La température se trouve dans le fichier temperature_readings.json, il se trouve dans Data puis Sensor

  • 1- lancer le script: run help.sh
  • 2- ensuite lancer le script avec run nino.sh , avec run celie.sh et avec run matias.sh

Si toutes les températures sont négatives:

  • Cela n'est pas normal ça doit etre un oubli, mettre pour toues les valeurs remplacer le '-'' par '+'

Si une seule température est négative:

  • changer la troisième valeur avec x telle que 0 = température1*x³ + 50
  • changer la deuxieme valeur avec y telle que 0 = température2*y-5 + x
  • changer la première valeur avec 0
